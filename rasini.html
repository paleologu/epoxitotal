Ce sunt rasinile epoxidice?

add remove

Rasinile epoxidice sunt materiale polimerice, bicomponente, care la inceput se afla in stare lichida, apoi in baza unei reactii chimice a celor doua componente, se transforma intr-un material solid.

Cat de rezistente sunt pardoselile din rasini?

add remove

Rasinile sunt rezistenta la coroziune si sunt un foarte bun protextor impotriva apei si umiditatii. Cea mai cunoscuta utilizare a rasinilor este in domeniul pardoselilor deoarece este foarte rezistent, igienica si usor curatabila.

In ce situatii devin necesare?

add remove

Pardoseaua din rasini epoxidice este pretabila pentru spatii industriale cu trafic intens (fabrici, depozite, hale, service auto, spitale), deoarece protejeaza betonul impotriva actiunilor chimice si mecanice.

Rasinile epoxidice care sunt ideale pentru industria unde este contact cu agentii chimici foarte puternici si rasinile poliuretanice care sunt in proportie de 80% elastice si sunt specifice industriei in care exista socuri mecanice semnificative.
Materialele folosite de catre Epoxitotal in aplicarea pardoselilor sunt certificate TUV Austria si verificate in laborator inainte de aplicare. Deasemenea, sistemul de management este certificat ISO 9001. Pentru fiecare aplicare se ofera certificate de calidtate si garantie.
